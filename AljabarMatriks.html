Aljabar Matriks#

#Penjelasan Mengenai Matriks ##Pengertian Matriks: Matriks adalah susunan bilagan real(atau elemen-elemen) yang disusun dalam baris dan kolom sehingga membentuk persegi ataupun persegi panjang. Setiap elemen memiliki posisi unik dalam matriks, ditentukan oleh baris dan kolomnya. Misalnya, matriks 2x3 memiliki 2 baris dan 3 kolom. suatu matriks biasa diberi nama dengan menggunakan huruf kapital (seperti A, B dll) dan elemennya dinyatakan dengan huruf kecil.

Berikut Contoh Matriks#

\[\begin{split} \begin{align*} A= \begin{bmatrix} 2&3&2\\ 4&1&-3\\ -1&-2&2 \end{bmatrix} && B= \begin{bmatrix} 2&4&7\\ 6&-3&2\\ 5&-2&7 \end{bmatrix} && C= \begin{bmatrix} 3&4\\ -2&4\\ 2&5 \end{bmatrix} \end{align*} \end{split}\]

##1.Operasi Penjumlahan Matriks :

Operasi ini melibatkan penambahan elemen-elemen yang berada pada posisi yang sama dari dua matriks yang memiliki ordo yang sama. Artinya kedua matriks ini harus memiliki baris dan kolom yang sama.

CONTOH PENJUMLAHAN MATRIKS A + B :

\[\begin{split} \begin{align*} A= \begin{bmatrix} 2&3&2\\ 4&1&-3\\ -1&-2&2 \end{bmatrix} && B= \begin{bmatrix} 2&4&7\\ 6&-3&2\\ 5&-2&7 \end{bmatrix} \end{align*} \end{split}\]
\[\begin{split} \begin{align*} A + B = \begin{bmatrix} 2+2&3+4&2+7\\ 4+6&1+(-3)&-3+2\\ -1+5&-2+(-2)&2+7 \end{bmatrix} = \begin{bmatrix} 4&7&9\\ 10&-2&-1\\ 4&-4&9 \end{bmatrix} \end{align*} \end{split}\]

BERIKUT CODE PENJUMLAHAN MATRIKS DENGAN MENGGUNAKAN ATAU MEMANGGIL LIBRARY NUMPY:

##2.Operasi Pengurangan Matriks:

Operasi ini melibatkan pengurangan elemen-elemen yang berada pada posisi yang sama dari dua matriks yang memiliki ordo yang sama. Artinya kedua matriks ini harus memiliki baris dan kolom yang sama.

CONTOH PENGURANGAN MATRIKS:

\[\begin{split} \begin{align*} A= \begin{bmatrix} 2&3&2\\ 4&1&-3\\ -1&-2&2 \end{bmatrix} && B= \begin{bmatrix} 2&4&7\\ 6&-3&2\\ 5&-2&7 \end{bmatrix} \end{align*} \end{split}\]
\[\begin{split} \begin{align*} A - B = \begin{bmatrix} 2-2&3-4&2-7\\ 4-6&1-(-3)&-3-2\\ -1-5&-2-(-2)&2-7 \end{bmatrix} = \begin{bmatrix} 0&-1&-5\\ -2&4&-5\\ -6&-0&-5 \end{bmatrix} \end{align*} \end{split}\]

BERIKUT CODE PERKALIAN MATRIKS DENGAN MENGGUNAKAN ATAU MEMANGGIL LIBRARY NUMPY:

##3.Operasi Perkalian Matriks:

Operasi ini melibatkan perkalian elemen-elemen matriks. Untuk melakukan perkalian matriks A dan B, jumlah kolom dalam matriks A harus sama dengan jumlah baris dalam matriks B. Contoh matriks A memiliki ordo atau ukuran mxn, maka matriks B harus memiliki ordo atau ukuran nxp. Sehingga hasil dari perkalian tersebut akan menghasilkan matriks baru dengan ukuran mxp. Hasilnya akan memiliki jumlah baris yang sama dengan matriks A dan jumlah kolom yang sama dengan matriks B.

CONTOH PERKALIAN MATRIKS:

\[\begin{split} \begin{align*} A= \begin{bmatrix} 2&3&2\\ 4&1&-3\\ -1&-2&2 \end{bmatrix} && B= \begin{bmatrix} 2&4&7\\ 6&-3&2\\ 5&-2&7 \end{bmatrix} \end{align*} \end{split}\]
\[\begin{split} \begin{align*} A X B = \begin{bmatrix} 2(2)+3(6)+2(5) &2(4)+3(-3)+2(-2)&2(7)+3(2)+2(7)\\ 4(2)+1(6)+(-3)(5) &4(4)+1(-3)+(-3)(-2) &4(7)+1(2)+(-3)(7)\\ -1(2)+(-2)(6)+2(5) &-1(4)+(-2)(-3)+2(-2) &-1(7)+(-2)(2)+2(7) \end{bmatrix} \end{align*} \end{split}\]
\[\begin{split} \begin{align*} AB= \begin{bmatrix} 32&-5&34\\ -1&19&9\\ -4&-2&3 \end{bmatrix} \end{align*} \end{split}\]

BERIKUT CODE PERKALIAN MATRIKS DENGAN MENGGUNAKAN ATAU MEMANGGIL LIBRARY NUMPY:

##4.Operasi Transpose Matriks

Transpose matriks adalah matriks baru yang diperoleh dengan cara menukar elemen-elemen baris menjadi elemen kolom atau sebaliknya. Jika sebuah matriks berukuran M X N, maka transpose matriks tersebut memiliki ordo N X M.

CONTOH TRANSPOSE MATRIKS:

\[\begin{split} \begin{align*} C= \begin{bmatrix} 3&4\\ -2&4\\ 2&5 \end{bmatrix} && CT (C Transposre)= \begin{bmatrix} 3&-2&2\\ 4&4&-5 \end{bmatrix} \end{align*} \end{split}\]

BERIKUT CODE UNTUK MENG-TRANSPOSE MATRIKS DENGAN MENGGUNAKAN ATAU MEMANGGIL LIBRARY NUMPY:

import numpy as np

c=np.array([[3,4],[-2,4],[2,5]])

c_tran = c.transpose()
print('matrik c sebelum di transpose=\n', c)
print('matrik c setelah di transpose=\n',c_tran)
print('ordo C_transpose adalah',c_tran.shape)
z=c_tran@c
print('Z = C_transpose x C =\n',z)
print('ordo z adalah',z.shape)
p=c@c_tran
print('P = C x C_tranpose =\n',p)
print('ordo P adalah',p.shape)
print("Dapat disimpulkan bahwa C_transpose X C tidak sama dengan C X C_transpose")

#Program untuk Mengalikan Matriks dengan Menggunakan Perulangan (Loop) For :

def perkalian(matrix1, matrix2):
    if len(matrix1[0]) != len(matrix2):
        print("Perkalian matriks tidak dapat dilakukan karena jumlah kolom dalam matriks pertama tidak sama dengan jumlah baris dalam matriks kedua")
        return None
    result = [[0 for _ in range(len(matrix2[0]))] for _ in range(len(matrix1))]
    for i in range(len(matrix1)):
        for j in range(len(matrix2[0])):
            for k in range(len(matrix2)):
                result[i][j] += matrix1[i][k] * matrix2[k][j]
    return result

def tampilan_hasil(matrix):
    for baris in matrix:
        print(baris)

print ("Berikut adalah contoh matriks")

matriks 1=([[2,4,7],[6,-3,2],[5,-2,7]])
""")
matriks1=([[2,3,2],[4,1,-3],[-1,-2,2]])
matriks2=([[2,4,7],[6,-3,2],[5,-2,7]])
test = perkalian(matriks1, matriks2)
print("\nHasil perkalian matriks:")
tampilan_hasil(test)
