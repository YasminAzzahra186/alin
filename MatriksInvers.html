Matriks invers#

Matriks yang jika dikalikan dengan matriks asalnya, akan menghasilkan metriks identitas. Dan sering digunakan dalam pemecahan sistem persamaan linier.

AA^-1 = A^-1 x A = I

I adalah matriks identitas yang memiliki elemen-elemen diagonal bernilai 1 dan elemen-elemen lainnya bernilai 0. Matriks invers ini hanya ada untuk matriks persegi yang tidak singular, determinannya tidak nol. Contoh :

  • Matriks dengan ordo(2 x2):

\[\begin{split} \begin{equation*} A= \begin{bmatrix} a&b\\ c&d \end{bmatrix} \end{equation*} \end{split}\]
\[\begin{split} \begin{align*} A^-1\ =\frac{1}{ad-bc} \begin{bmatrix} d&-b\\ -c&a \end{bmatrix} \end{align*} \end{split}\]

  • Matriks dengan ordo (3 x3):

\[\begin{split} \begin{equation*} A= \begin{bmatrix} a&b&c\\ d&e&f\\ g&h&i\ \end{bmatrix} \end{equation*} \end{split}\]
\[ \begin{equation*} A^-1=\frac{1}{aei + bfg + cdh - gec - hfa - idb} Adj A \end{equation*} \]

Contoh matriks dengan ordo (2 x2):

\[\begin{split} \begin{equation*} A= \begin{bmatrix} 2&-1\\ -3&2 \end{bmatrix} \end{equation*} \end{split}\]

Untuk mencari inversnya dapat dilakukan dengan mencari determinannya terlebih dahulu, menggunakan rumus di bawah ini.

det A = ad - bc

Berikut penyelesaian determinannya:

det A = (2 x 2) - (-1 x (-3)) = 4 - 3 = 1

Setelah ditemukan nilai determinannya, langkah selanjutnya yaitu masukkan nilai determinan pada rumus invers matriks dengan ordo (2 x 2)

\[\begin{split} \begin{equation*} A^-1=\frac{1}{det A} \begin{bmatrix} d&-b\\ -c&a \end{bmatrix} \end{equation*} \end{split}\]

Berikut penyelesaian Matriks invers nya:

\[\begin{split} \begin{equation*} A^-1=\frac{1}{1} \begin{bmatrix} 2&1\\ 3&2 \end{bmatrix} = \begin{bmatrix} 2&1\\ 3&2 \end{bmatrix} \end{equation*} \end{split}\]

###Sifat-sifat Matriks invers

  1. Hanya dimiliki oleh matriks persegi karna memiliki kolom dan baris yanng sama.

  2. Hanya ada matriks non-singular. Determinannya tidak sama dengan 0.

  3. Unik : Setiap matriks hanya memiliki satu matriks invers

  4. Invers dari invers : Jika Z memiliki invers, maka inversnya adalah matriks Z itu sendiri. \(\begin{align*}(Z^-1)^-1=Z\end{align*}\)

  5. Invers dari hasil kali : Jika Y dan Z adalah matriks yang memiliki invers, maka hasil kali AB juga memiliki invers, dan invernya adalah hasil kali invers secara terbalik. \(\begin{align*}(YZ)^-1=(Z^-1)Y^-1\end{align*}\)

  6. Invers dari tranpose : jika Z merupakan matriks persegi yang memiliki invers dari tranposenya adalah tranpose dari inversnya. \(\begin{align*}(Z^T)^-1=(Z^-1)^T\end{align*}\)

  7. Invers dari matriks identitas : Invers dari matriks identitas I adalah I itu sendiri

###Mencari Invers Dengan Eliminasi Gauss Jordan Elimination

Langkah-langkah Eliminasi Gauss Jordan adalah

  1. Membuat elemen diagonal utama menjadi 1. Hal ini dilakukan uuntuk mempermudah meng-nol-kan elemen yang bukan diagonal utama.

  2. Membuat nol di selain diagonal utama. Hal ini dilakukan dengan menggunakan operasi baris seperti

  • Menukarkan posisi baris satu dengan baris yang lain (RowSwap)

  • Mengkalikan baris dengan bilangan selain nol (RowScale)

  • Menambahkan kelipatan baris ke baris yang lain (RowAdd)

  1. Mengulangi proses untuk setiap kolom

Berikut contoh matriks A: $\( \begin{equation*} A= \begin{bmatrix} 6&5\\ 8&10 \end{bmatrix} \end{equation*} \)$

Eliminasi Gauss Jordan dilakukan dengan menempatkan matriks A di sebelah kiri dan matriks identitas(I) di sebelah kanan. Kedua matriks ini dipisahkan dengan garis putus-putus. Seperti berikut:

\[\begin{split} \begin{equation*} A= \begin{bmatrix} 6&5& | &1&0 \\ 8&10& | &0&1 \end{bmatrix} \end{equation*} \end{split}\]

Berikut penyelesaian menggunakan Eliminasi Gauss Jordan Elimination:

  1. Bagilah baris 1 dengan 6 [R1 = (1/6) R1]

\[\begin{split} \begin{equation*} A= \begin{bmatrix} 1&5/6& | &1/6&0 \\ 8&10& | &0&1 \end{bmatrix} \end{equation*} \end{split}\]

  1. [R2 = R2 - 8R1]

\[\begin{split} \begin{equation*} A= \begin{bmatrix} 1&5/6& | &1/6&0 \\ 0&10/3& | &-4/3&1 \end{bmatrix} \end{equation*} \end{split}\]

  1. [R2 = (3/10)R2]

\[\begin{split} \begin{equation*} A= \begin{bmatrix} 1&5/6& | &1/6&0 \\ 0&1& | &-2/5&3/10 \end{bmatrix} \end{equation*} \end{split}\]

  1. [R1 = R1 - (5/6)R2]

\[\begin{split} \begin{equation*} A= \begin{bmatrix} 1&0& | &1/2&-1/4 \\ 0&1& | &-2/5&3/10 \end{bmatrix} \end{equation*} \end{split}\]
import numpy as np

def invers_matriks_2x2(A):
    # Periksa apakah matriks A dapat diinverskan
    if np.linalg.det(A) == 0:
        print("Matriks tidak dapat diinverskan karena determinan = 0.")
        return None

    # Buat matriks identitas berukuran 2x2
    I = np.eye(2)

    # Gabungkan matriks A dan matriks identitas
    aug_matrix = np.hstack((A, I))

    # Lakukan operasi baris elementer untuk mengubah matriks A menjadi matriks identitas
    for i in range(2):
        # Bagi baris ke-i dengan elemen diagonal
        pivot = aug_matrix[i, i]
        aug_matrix[i, :] /= pivot
        # Kurangi baris lain dengan kelipatan baris ke-i
        for j in range(2):
          if j != i:
            factor = aug_matrix[j, i]
            aug_matrix[j, :] -= factor * aug_matrix[i, :]

     # Matriks invers berada di sisi kanan matriks augmentasi
    invers_A = aug_matrix[:, 2:]

    return invers_A

print(invers_A)

def RowSwap(A,k,l):
# =============================================================================
#     A is a NumPy array.  RowSwap will return duplicate array with rows
#     k and l swapped. (menukarkan persamaan)
# =============================================================================
    m = A.shape[0]  # m is number of rows in A (jumlah baris A)
    n = A.shape[1]  # n is number of columns in A (jumlah kolom A)

    B = np.copy(A).astype('float64')

    for j in range(n):
        temp = B[k][j]
        B[k][j] = B[l][j]
        B[l][j] = temp

    return B

def RowScale(A,k,scale):
# =============================================================================
#     A is a NumPy array.  RowScale will return duplicate array with the
#     entries of row k multiplied by scale. (mengalikan entry baris A dengan skala)
# =============================================================================
    m = A.shape[0]  # m is number of rows in A
    n = A.shape[1]  # n is number of columns in A

    B = np.copy(A).astype('float64')

    for j in range(n):
        B[k][j] *= scale

    return B

def RowAdd(A,k,l,scale):
# =============================================================================
#     A is a numpy array.  RowAdd will return duplicate array with row
#     l modified.  The new values will be the old values of row l added to
#     the values of row k, multiplied by scale. (menambahkan nilai baris k dikalikan dengan skala)
# =============================================================================
    m = A.shape[0]  # m is number of rows in A
    n = A.shape[1]  # n is number of columns in A

    B = np.copy(A).astype('float64')

    for j in range(n):
        B[l][j] += B[k][j]*scale

    return B
